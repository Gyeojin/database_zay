## 주요 특징
* 메인페이지 / 상품 리스트 / 상품 상세페이지 / 리뷰 페이지 작업   
* 순수 jQuery 사용으로 기능 연습/구현
* jQuery 플러그인 사용으로 다양한 슬라이드 구현   
* Community 페이지 넘버링 / 글 작성+수정+삭제 구현   
* like, unlike 카운팅 기능 구현
